# Smart HVAC System Simulation (MATLAB)

## Overview
This project models and simulates a smart HVAC system using MATLAB.
The system integrates mechanical actuator dynamics, airflow behavior,
thermal energy balance, and indoor air quality (CO₂) modeling.

## Subsystems
- Actuator dynamics for dampers and fans
- Airflow and ventilation response
- Indoor temperature evolution
- CO₂ concentration and control response

## Repository Structure
- `/scripts` – MATLAB subsystem models
- `/figures` – Simulation results and plots

## Where to Start
1. `scripts/WholeSystem.m`
2. `figures/` for system response plots
